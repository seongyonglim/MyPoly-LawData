# 개발자용 문서

> 백엔드 개발자가 참고할 기술 문서들

## 문서 목록

### 핵심 설계 문서
- **`db-design-final.md`** (최신) - 대량 데이터 분석 결과 반영한 최종 DB 설계
- **`api-field-mapping.md`** - API 필드명 매핑
- **`crawling-strategy-updated.md`** - 데이터 크롤링 전략 (일일/주간 배치)

### API 분석 문서
- **`api-crawling-results.md`** - API 실제 응답 데이터 상세
- **`bulk-analysis-results.md`** - 대량 데이터 분석 결과
- **`api-adequacy-review.md`** - API 충분성 검토 (추가 API 불필요 확인)
- **`api-review.md`** - API 검토 결과
- **`api-test-results.md`** - API 테스트 결과
- **`api-authentication.md`** - API 인증키 관리

### 화면 분석 문서
- **`figma-screen-analysis.md`** - 의안 상세 화면 분석
- **`member-detail-analysis.md`** - 의원 상세 화면 분석

### AI 코드 분석
- **`ai-code-analysis.md`** (최신) - AI 디렉토리 코드 분석 (크롤링 + 요약)

### 구현 가이드
- **`implementation-checklist.md`** - 구현 체크리스트
- **`data-collection-strategy.md`** - 데이터 수집 전략 (구버전)
- **`feature-mapping.md`** - 기능-API 매핑
- **`figma-sharing-guide.md`** - Figma 공유 가이드

### 기타
- **`db-design.md`** - DB 설계 (구버전, `db-design-final.md` 참고)
- **`implementation-summary.md`** - 구현 가능성 종합 (구버전)
